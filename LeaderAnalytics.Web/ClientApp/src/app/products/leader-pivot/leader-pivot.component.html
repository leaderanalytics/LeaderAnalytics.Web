Leader Pivot

Pivot control for analyzing hierarchical data

Leader Pivot is a low-level library that provides logic for grouping data and building a sized cell matrix. It can be used as base library to build a pivot control for a specific UI such as HTML, Blazor, or XAML. One implementation currently exists for Blazor.

  • Easy to implement and configure
  • Drag and drop dimensions across axis
  • User configurable measures
  • Four color themes provided, customize or create your own.
