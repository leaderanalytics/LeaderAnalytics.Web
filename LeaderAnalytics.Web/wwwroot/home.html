Vintages

Our flagship data aggregation and reporting tool. The only product that allows the user to fully utilize FRED vintage data in regression models.

  • Easily align observations by Vintage Date or Observation Date
  • Use vintage data to create forecasts based on information that was publicly available on a given date
  • Avoid "peeking" errors that result from using data revised in a future period
  • Compare forecasts across multiple models
  • Save and recreate models using a local database that ensures repeatable results
  • Acquire data from multiple sources, including the Federal Reserve Bank of St. Louis (FRED)
